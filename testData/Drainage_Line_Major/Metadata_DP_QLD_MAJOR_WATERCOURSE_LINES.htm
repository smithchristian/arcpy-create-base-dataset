Description

Abstract / Description:
This dataset displays the major watercourse lines covering the State of Queensland.Major watercourses are those as classified by Geoscience Australias 250000 Topographic Data as being Australias signigicant watercourses. In this dataset a small number of other watercourses have also been classified as major watercourses so that all sub-drainage basin areas in Queensland have at least one major watercourse within them. This dataset is a subset of the Watercourse Lines - Queensland dataset. The major watercourses are connected and flow directed to create a linear network for hydrological modelling. Features are attributed with perenniality and names where available.Features within the Watercourse Lines - Queensland dataset of which this dataset is a subset of, have been captured or updated from the best available imagery with an attribute within the data describing the source and reliability. Data sources include: - Queensland Orthophotography - Satellite Imagery- Geoscience AustraliaNames have been sourced from:- Queensland Place Names Database- Geoscience Australia 250000 Topographic Data- Existing Topographic and Cadastral Mapping- Survey Plans- Field checks
Purpose:
To provide digital data of major watercourse features for use in Land Administration, Emergency Management, Monitoring of Climate Change, Hydrological Modelling, and Topographic Mapping and in the production of Navigational and Web Based Mapping applications.

Additional Information:
This dataset has been compiled as part of a collaborative program between the Queensland Government and Geoscience Australia to upgrade the states hydrographic data for inclusion in the national geospatial fabric.Features have been progressively updated by drainage basin using imagery. Those basins along the east coast from the New South Wales border to Cooktown are captured at 25000 mapping specifications while those in the Murray Darling Basins, the Fitzroy and Burdekin and those basins flowing into the Gulf of Carpentaria or within Cape York Peninsula are captured at 50000 mapping specifications. The balance of the state is at 100000 mapping specifications. The Coleman, Flinders, Gilbert, Holroyd, Mitchell, Norman, Settlement and Staaten Basins that flow into the Gulf of Carpentaria and the Jeannie and Normanby Basins that flow into the Coral Sea are currently being updated or will be in the near future and will be incorporated into this data series when complete. 250000 data from Geoscience Australia currently is used in these areas to fill the gap and will be progressively replaced with the updated data.

History:

Lineage statement:
This dataset is a subset of the Watercourse Lines - Queensland dataset.The dataset was compiled by extracting those features within the above dataset that had a classification of Major from the Hierarchy attribute column. The Watercourse Lines - Queensland dataset that this dataset has been compiled from was scanned from topographic repromats, captured by photogrammetry, digitised from satellite imagery or taken from Geoscience Australias 250000 topographic datasets with reliability dates ranging from 1978 to 2005. This original data has been spatially updated between 2010 and the start of 2014 using the best available imagery over the area, 10cm - 60cm orthophotography along much of the east coast, 2.5m satellite imagery over the interior, the Gulf of Carpentaria and Cape York Peninsula.Original data is compared to current imagery and updated (moved to correct position, deleted as the feature no longer exits or added as the feature is new). Linear features within the data have been concatenated and segmented at intersections and flow directed down-stream to assist with hydrological modelling. Watercourse connectors flow through waterbodies to continue linear networks. Features have been attributed with name and name source where available, perenniality and hierarchy.As part of a collaborative agreement between the Queensland Government and Geoscience Australia, both organisations undertook to update the features within allocated drainage basins. Those drainage basins updated by the Queensland Government were updated using Microstation Geographics V8 in MGA94 (GDA94) before the data was transformed to Latitude/Longitude (GDA94). Geoscience Australia undertook their update process using ESRI ArcGIS V9 or V10. The Queensland Government is responsible for merging all basins into a whole of state dataset stored within an ESRI Geodatabase that agrees with the Geoscience Australia Topographic Database Schema in Latitude/Longitude (GDA94).This downloadable dataset is an ESRI shape version of the ESRI Geodatabase. Some columns names have been altered to allow for the shortened column name description in the shape format.Feature Types:Major Watercourse - A line respresnting the significant watercourses within Queensland.Attributes of features have been determined from numerous sources, including: topographic mapping, cadastral mapping, regional mapping, survey plans, Queensland Place Names Database, other departmental published maps or directories.Attributes:Feature - Unique feature type that identifies the topographic feature e.g. Major Watercourse.Name - The name of the feature (if available)Alt_Name - An alternate name of the feature (if available)Indigenous - An indigenous name of the feature (if available)Flow - (Perennial or Non Perennial - If perennial then water flows most of the time) Based on Geoscience Australias 100000 Topographic Data defined perennial watercourses.SourceDate - Reliability date of the source informationSource - The imagery, map or data that the spatial location of the feature was obtained from.AttDate - Reliability date of the document or authority that the name of the feature was acquired from.AttSource - The source (imagery, map or data) used for attributing (naming) of the feature. If no name is present then the attribute source will be the imagery, map or data used to define the feature type e.g. culvertPlanAccur - Planimetric Accuracy - The standard deviation of the horizontal positional accuracy.TextNote - A comment field
